This article discusses the requirements, speed settings, cruise target speed settings, and test suggestions for the L2GB ADRS speed limit system. Key points include: ​
System requirements : The system should identify the current road speed limit through maps and traffic signs and display the current road limit speed and the system's cruise target speed on the dashboard.​
Speed settings : For A - class roads, the driver - set maximum speed should not exceed 120km/h; for B - class roads, it should not exceed 100km/h. When the system is activated in corresponding road environments, the driver - set maximum speed should not exceed the highest limit speed within the ODC.​
Cruise target speed settings : The cruise target speed can be determined by the driver - set speed or the system - determined current road limit speed. When the road limit speed changes, the adjustment of the cruise target speed follows specific rules.​
Speed setting default offset : If the system allows the driver to set a default offset, the offset should not make the cruise target speed greater than the system - determined current road limit speed.​
Driver speed setting : The system should provide a way for the driver to adjust the set speed, and the setting method is described in document 3 DA HMI / User Interactions.​
System speed control and warning : Except when the driver's acceleration input is greater than the system - induced acceleration, the vehicle speed should not exceed the cruise target speed. When the vehicle speed exceeds the road limit speed, the system should issue continuous optical and acoustic signals.​
