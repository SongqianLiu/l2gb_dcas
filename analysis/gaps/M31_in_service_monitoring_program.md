## EU Requirement (R.171 §7.1)

> The manufacturer shall establish and maintain a post-market monitoring program to assess whether the DCAS system continues to perform in accordance with the approved requirements during in-service operation. The program shall include:
> - Monitoring of real-world operational data
> - Analysis of safety-critical occurrence patterns
> - Processes for corrective action when systematic issues are identified

This is a formal in-service monitoring (ISM) obligation — not just passive recall management but active performance monitoring of the system in real-world use.

---

## China L2GB Status

The CN GB draft does not contain an equivalent in-service monitoring program requirement for L2GB DCAS systems.

**This requirement is not found in the CN L2GB standard.** It is an EU-specific post-market surveillance obligation aligned with the EU's General Safety Regulation (GSR2) and the broader post-market monitoring framework for vehicle safety systems.

---

## Scope of Required Program

An in-service monitoring program for DCAS typically includes:
1. **Data collection**: on-vehicle event recording (DCAS activations, deactivations, EOR events, intervention events)
2. **Aggregation**: fleet-level data aggregation for statistical analysis
3. **Review cadence**: periodic (e.g., quarterly) review of collected data for safety trends
4. **Corrective action**: defined process for escalating identified issues to software updates or recalls
5. **Documentation**: program plan submitted to the TA as part of type approval

## Required Closure Actions

1. Develop an in-service monitoring program plan for G1.3 EU DCAS
2. Define what data will be collected from each vehicle and at what frequency (consider telemetry infrastructure already in place for CN market)
3. Define the review cadence and responsible team
4. Define thresholds for corrective action (e.g., if incident rate exceeds X per million km)
5. Submit the program plan to the EU TA as part of the type approval dossier
6. Link the ISM program outputs to the annual reporting obligation (M30)
